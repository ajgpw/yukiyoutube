単項式と多項式の乗法

これは、2年生で学んだ「文字を含んだ乗法・除法」に毛が生えただけですね！

単項式 … 項(元)が1つ ex) 6, a, x2, 2x, xy, xy2 などなど

多項式 … 項(元)が2つ以上 ex) 2+3, 2-b, x+y, x2-2xy+y2 など

そして、単項式×多項式ということは、

2x(3x+4y)=6x2+8xy

というふうに1つの「項」自体が少し複雑になっただけで『分配法則』さえ知っていれば、全く同じ手順ですね！

多項式と多項式の乗法

多項式どうしの乗法(掛け算)も、「分配法則」さえ知っていれば計算(展開)できますね、

a(x+y)=ax+ay

(a+b)(x+y)=ax+ay+bx+xy

(a+b+c)(ax+ay+az+bx+by+bz+cx+cy+cz)

● (a+b)(c+d)(x+y) = acx+acy+adx +ady+bcx+bcy+bdx+bdy

このような「ルール(展開方法)」でしたね！

ですが、今後は特に(●+△)(●+□) のような形が、多く出てきます！すなわち、 2つの(カッコ) の中の先頭の項が「同じ」ものですね！

例：

(x-4)(x+5) = x2+x-20

というわけで、1つ目の公式の完成です。

公式：

(x+a)(x+b) = x2+(a+b)x+ab

(x+a)2 = x2+2ax+a2

(x-a)2 = x2-2ax+a2

(x+a)(x-a) = x2-a2

例：

(2x+3)(2x+4) = 4x2+14x+12

(3x-1)(3x+2) = 9x2+3x-2

(x+2y)(x+3y) = x2+5xy+6y2

因数分解

素因数分解や因数分解の意味や必要性やイメージはもう2年生課程で見てきましたので、ここでは省略しますね。

〈レベル1〉共通因数でくくる

a(a+1)を展開すると → a2+a でしたね因数分解は、その逆の作業でしたからa2+a を因数分解すると、 = a(a+1) ですね

例：

4ax+8ay = 4a(x+2y)

8xy2-6xy = 2xy(4y-3)

〈レベル2〉 展開公式の逆

展開公式は

  1. (x+a)(x+b) = x2+(a+b)x+ab
  2. (x+a)2 = x2+2ax+a2
  3. (x-a)2 = x2-2ax+a2
  4. (x+a)(x-a) = x2-a2

でしたね

例：

x2+3x-28 = (x+7)(x-4)

x2-9x+18 = (x-3)(x-6)

x2+10x+25 = (x+5)(x+5) = (x+5)2

x2-16x+64 = (x-8)(x-8) = (x-8)2

x2+5xy+6y2 = (x+2y)(x+3y)

x2+xy-20y2 = (x+5y)(x-4y)

〈レベル3〉 置き換え

例：

(x+y)2-2(x+y)-3 = ((x+y)-3)((x+y)+1) = (x+y-3)(x+y+1)

〈レベル4〉 1度展開

例：

(x-1)(x-2)-6 = x2-3x+2-6 = x2-3x-4 = (x-4)(x+1)

〈レベル5〉 部分因数分解

例：

ax-by-ay+bx = a(x-y)-b(x-y) = (a-b)(x-y)

(x+y2)-(a+y2) = (x-a)(x+y)

〈レベル6〉 〇2-△2=(〇+△)(〇-△)

常に 〇2-△2 を意識しましょう！

例：

(2x+y)2-1 = (2x+y+1)(2x+y-1)

ポイントまとめ

  • 共通因数でくくる
  • 展開公式の逆
  • 置き換え（カッコを付けて「1文字」と見る）
  • 1度展開
  • 部分因数分解
  • 〇2-△2を常に意識する

例題

例1：

(x + 5)(x + 3) = x2 + 8x + 15

例2：

(x + 1)2 - 5(x - 1) - 16

= x2 + 2x + 1 - 5x + 5 - 16

= x2 - 3x - 10

= (x - 5)(x + 2)

例3：

(2a - b)x - 2(b - 2a)y

= (2a - b)x + 2(2a - b)y

= (2a - b)(x + 2y)
